# Elrond smart contracts

Just playing with elrond smart contracts

## Egld pay

### Build

```shell
$ bash ./egld-pay/cmd/build.sh
```

### Test
```shell
$ cd ./egld-pay/sc && cargo test
```

### Generate new wallet

```shell
$ bash ./egld-pay/wallets/new.sh <WALLET_NAME>
```

### Deploy testnet

```shell
$ bash ./egld-pay/cmd/deploy.sh
```
